# Marketplace Client

## Overview
Handles sync, download, and install flows for published module/integration packages.

## Genel Bakis
Yayınlanan modül/entegrasyon paketlerinin senkronizasyon, indirme ve kurulum akışını yürütür.

## Features
- Family: `core`
- Version: `1.0.0`
- Admin routes: `true`
- API routes: `true`
- Views: `true`
- Lang files: `true`
- Migrations: `false`

## Dependencies
- None

## Repository
This module is packaged and released as an independent artifact in Flow marketplace workflows.
